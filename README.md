<h3>Collections Framework – Student Management System</h3>
<h4>ArrayList:</h4> 
Stores students dynamically (size can grow).

<h4> HashMap:</h4>
Search student quickly using ID — faster than looping.

<h4> Comparator:</h4>
Sort students by marks.

<h4> Set:</h4>
Automatically prevents duplicates.

<h4>Enhanced For Loop:</h4>
Cleaner than normal for loop.
